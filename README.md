# fitness-management
spring boot, jsp를 이용해서 만든 피트니스 고객 관리 시스템

<br/>

## 💻 프로젝트 소개
관리자와 이용자가 상호작용하며
<br/>
이용자가 로그인 후 프로그램을 신청하고, 관리자는 회원관리와 프로그램관리 등을 수행하는 프로그램입니다.
<br/>
<br/>
